Encargo 05 - Modelado 3D

Primeros Bocetos

Con el fin de crear un acolchado a partir de caucho reciclad, comenzamos a pensar como se puede llevar a cabo una estructura que mantenga las propiedades del caucho y que sea resistente. Nos centramos en diferentes estructuras que se pueden ver en la naturaleza, llamándonos la atención la estructura de los panales de abeja. Para estos bocetos tomamos en cuenta solo el diseño de la forma, dejando de lado la tecnica para lograr esta estructura, centrandosnos en la técnica de construcción pero sin dejar de lado el diseño comenzamos a crear diferentes iteraciones.

Actual Hipotesis de Diseño

Actualmente postulamos como diseño de nuestro producto esta estructura, manteniendo las formas hexagonales para generar flexibilidad y resistencia a los impactos, este modelo lo diseñamos pensando en como lo podríamos llevar a cabo, fue pensado para poder darle esta forma al caucho a través de moldes con estas formas. Nuestros siguientes pasos para poner a prueba este diseño es comenzar a experimentar con diferentes grosores y materiales para lograr el producto que buscamos lograr.
