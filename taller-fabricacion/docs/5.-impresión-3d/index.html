05. Impresión 3D

Impresión 3D

Estas maquinas son cada vez más populares en la población gracias a la considerable disminución de sus precios en los últimos años, las maquinas que antes parecían del futuro con precios muy elevados hoy las podemos encontrar por valores cercanos a los USD 400 en sus modelos básicos.

EL funcionamiento de estas maquinas se basa en la producción de objetos a través de capas, determinados por un código, el cual tiene un nombre de Código G, existen diferentes programas que analizaremos para obtener esto.

Existen varios tipos de impresoras, desde plástico hasta hormigón y el avance de esta tecnología va a un paso sorprendentemente rápido, acá hay un ejemplo de como se están comenzando a construir casas con impresoras 3D para que tengamos una idea de hasta donde se puede llegar con esta tecnología.

En esta sección nos centraremos principalmente en dos tipos de impresión, las más comunes que vendrían siendo las impresiones en PLA y las impresora SLA.

Tutorial

Para realizar los siguientes pasos puedes utilizar los archivos de cualquier modelo 3D. Acá unos archivos que pueden ser útiles:

Descargar
  • Primero debemos obtener nuestro archivo en formato .stl para esto se puede usar varios programas de diseño, acá mostraremos como realizarlo en Rhino 7, utilizando el archivo Quilla_TwinTip.obj

Impresión 3D PLA+

  • Para comenzar debemos descargar el Software Cura, esté programa Open Source nos ayudara a obtener nuestro Código G para poder realizar la impresión, el link de descarga es el siguiente: https://ultimaker.com/es/software/ultimaker-cura, una vez instalado el programa configuramos la impresora que utilizaremos.
  • Seleccionamos el material que utilizaremos, en este caso PLA+ y también seleccionamos el diámetro del cabezal que usaremos, en este caso 0.4mm
  • Abrimos nuestro archivo .stl, en este caso usaremos el archivo Quilla_TwinTip.stl.
  • Configuramos los ajustes y parámetros que queremos modificar, esto va a depender de que tan resistente queremos que sea nuestra pieza y el tiempo de impresión que queremos lograr, cuando tenemos nuestros parámetros establecidos seleccionamos Slice
  • Con el comando Preview podemos ver como seria el proceso de impresión y la estructura interna de nuestro objeto.

Cabezales

Para este tipo de impresiones existe usan relación directa entre detalles de la muestra y tiempo de impresión, uno de los factores clave en esto es el diámetro de los cabezales, a menor diámetro más detalle pero a la vez mas tiempo de impresión y en el caso contrario de los cabezales de mayor diámetro los cuales crean formas con menos detalles pero con un menor tiempo de impresión. Los cabezales más comunes son los con diámetro de 0.4mm. a continuación un video con una explicación más detallada de estos conceptos y como optimizar la impresión según estos parámetros.

Impresión 3D SLA

Esta técnica de impresión se basa en la fotopolimerización para formar cuerpos 3D, este proceso es básicamente la solidificación de resina a través de rayos UV, obteniendo figuras altamente precisas y con un acabado de mayor precisión que las maquinas más básicas.

Esta empresa es de las pioneras y de las más reconocidas en esta área, https://formlabs.com/, en la pagina detallan y especifican el proceso.

  • Para desarrollar los siguientes pasos debemos instalar el programa de Form Labs para simular la impresión en SLA. Link Descarga: https://formlabs.com/software/#preform. Después de instalar el programa abrimos nuestro archivo .stl, a modo de ejemplo nosotros abriremos Quilla_TwinTip.stl.
  • Luego generamos los soportes para que la impresión se mantenga en pie mientras se crea.
  • Con los soportes creados podemos observar y editar la composición de la impresión.
