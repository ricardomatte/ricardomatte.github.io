06. Diseño Superficies

Diseño a Partir de Superficies

En el siguiente tutorial realizaremos un diseño a partir de superficies en el software Fusion 360.

  • Creamos un Boceto con el comando Punto de Ajuste de Spline, creamos un dibujo sin importar las medidas pero si fijándonos en las proporciones.
  • Subimos el plano donde dibujaremos la parte superior de nuestro diseño.
  • Creamos un dibujo desde la parte superior de nuestro diseño con el comando Punto de Ajuste de Spline.
  • Utilizamos el comando Solevación para unir ambos dibujos que realizamos.
  • Realizamos un dibujo con el comando Punto de Ajuste de Spline nuevamente.
  • Con el comando Extruir creamos intersecciones entre ambos cuerpos.
  • Con el comando Dividir Cuerpos creamos diferentes secciones en los cuerpos que dibujamos anteriormente.
  • Utilizamos el comando Recortar para quedarnos con las secciones que nos interesan.
  • Con el comando Parche cerramos nuestras superficies.
  • Con el comando Coser unimos las intersecciones de las superficies.
  • Luego aplicamos el comando Empale en las uniones que queremos “suavizar”.
  • Con el comando Vaciado creamos un vacío al interior del cuerpo que creamos, determinando los mm de las paredes internas.
  • Finalmente si activamos los bocetos que creamos en un comienzo y los modificamos podremos cambiar la forma de nuestro diseño.
